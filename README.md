## Sobre este projeto

Após adquirir novas habilidades e consolidar conhecimentos importantes na minha jornada, decidi organizar meus projetos em um portfólio acessível por meio do **GitHub Pages**.

A proposta deste repositório é:

- Demonstrar, na prática, minhas habilidades técnicas  
- Compartilhar soluções aplicadas a problemas reais de dados  
- Registrar minha evolução contínua na área  
- Servir como um ponto de apoio para recrutadores e profissionais interessados no meu trabalho
